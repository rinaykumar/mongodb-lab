# Mongodb demo

Download mongo from https://www.mongodb.com/try/download/community 

Download robo3t (Free version) from https://robomongo.org


If you have brew on Mac you can install like this:
https://docs.mongodb.com/manual/tutorial/install-mongodb-on-os-x/ 


If manual, unzip download. CD to directory then create a new empty database folder for storage.
run the binary `./bin/mongodb --dbpath ./path-to-datastore`
